Modelo de caja

El modelo de caja es especificamente de CSS, es decir es un módulo el cual define unas medidas de para los elementos de la página web a modo de bloques, tal como dice el título. Sus medidas son las siguientes:

  • Margen

    • Es la distancia entre dos elementosSe puede cambiar la distancia poniendo margin dentro de una etiqueta style y de un bloque o elemento concreto

  • Padding

    • Es la distancia de relleno entre el contenido del elemento y el borde. Se puede modificar esa distancia dentro de la etiqueta style y dentro de un bloque concreto.

  • Borde

    • Comno dice el nombre, es el borde del texto. Podemos modificar su grosor, color y forma poniendo "border" dentro de la etiqueta style y en un bloque concreto
