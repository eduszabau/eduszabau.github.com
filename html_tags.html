Insertar una tabla

Si queremos insertar una tabla en nuestra web,lo que debemos es usar la etiqueta table.
Tenemos que darle cabeza a la tabla por lo que usaremos la etiqueta thead.
Necesitamos usar la etiqueta tbody para dar cuerpo a la tabla.
Si queremos añadir líneas debemos usar la etiqueta th(tambien para tener el encabezado de la tabla).
Y td para añadir filas en el cuerpo.
