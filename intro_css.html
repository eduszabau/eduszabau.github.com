    • Selección por etiqueta

    Usamos la etiqueta style y dentro añadimos una etiqueta cualquiera y luego abrimos corchetes {} e introducimos el atributo que queramos cambiar con un valor determinado

    También podemos cambiar a nuestro gusto el color de las letras usando el atributo {color: rgb(255,0,0)} por ejemplo.
    Todos los que llevan la etiqueta p son amarillos.

    Selector de id

    Utiliza la misma estructura que la anterior pero usando "#". Además tendrás que definir una etiqueta con el atributo id="...".

    Este párrafo esta definido por un id.

    Uso por clase

    Es la misma estructura que las anteriores pero pones un "." en vez de un "#". Y además hay que definir uno o varias etiquetas con el atributo class="..."
    Todos los h2 estan definidos por la misma clase.
