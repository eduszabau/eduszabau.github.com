Boilerplate - EM

Es una plantilla de lenguajes de programación para crear sitios web que tenga compatibilidad con unos ordenadores. La cual destacan las siguientes:

  • Html Boilerplate
  • !Doctype hyml
  • html
  • head
  • meta charset
  • title
  • /head
  • body
  • /body
  • /html

Ahora pasaremos a los que no van ligados al Htm Boilerplate como pueden ser:

  1. h1 to h6
  2. br
  3. italic
  4. bold
  5. ul
  6. ol
  7. li

Usamos estas etiquetas para crear listas ordenadas como desordenadas

  2. Plátanos
  3. Huevos

DESORDENADAS
