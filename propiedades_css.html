Font Size(px)

El tamaño de fuente, conocido como "font size", lo usamos para cambiar el tamaño de la fuente. Para ello le usaremos el atributo font-size dentro de la etiqueta style, al cual le daremos un valor en píxeles.

Esto esta escrito con un tamaño de 20 px

Font Size (em)

Este atributo es usado también para cambiar el tamaño de la fuente, pero en vez de en píxeles, usamos "em"

Hemos aplicado el font-size con un 2em en este párrafo

Font-Style

Lo usamos para cambiar el estilo de la fuente. Usamos el atributo font-style dentro de la etiqueta style.

Podemos usar italic, la cual es la mas usada y es para poner en cursiva

Line-Height

Estre atributo conocido como altura entre líneas nos modifica la alatura entre líneas. Lo usamos poniendo el atributo line-height dentro de la etiqueta style

Este párrafo esta escrito con una altura de 40px.

Alinear Texto

Para alinear a cualquier lado que queramos debemos usar el atributo text-align dentro de la etiqueta style.

Si lo queremos a la derecha usamos text-align:right

Si lo queremos al centro lo mismo pero poniendo center

Y si lo queremos en la izquierda ponemos el mismo atributo pero añadimos left

Text-Decoration

Para la decoración de un texto usamos el atributo text-decoration dentro de la etiqueta style.
